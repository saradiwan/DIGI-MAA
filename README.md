# Todo App

A simple and responsive Todo application built using HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently with basic CRUD functionalities.

## Live Demo
https://jazzy-clafoutis-e21b0c.netlify.app/

## Features
- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Simple and clean user interface  
- Fully responsive design  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Netlify (Deployment)

## Project Purpose
This project was built as part of my learning journey to strengthen my JavaScript fundamentals and improve frontend development skills.

## What I Learned
- DOM manipulation  
- Event handling  
- Working with arrays and dynamic UI updates  
- Basic deployment using Netlify
